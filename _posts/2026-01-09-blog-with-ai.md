# AI와 함께 블로그 만들기

**비전공자 코딩 입문자의 Antigravity로 블로그 만들기**

---

## 🤔 Github 블로그가 뭐죠..그거 어떻게 하는 건데..

새싹 교육과정에서 만나게 된 전공자 분이 주최하시는 
**기술 블로그 포스팅** 스터디에 가입하게 되었다!  
티스토리와 벨로그 등 다양한 사이트가 있지만  
깃허브 블로그를 추천해주셔서 한번 만들어 보기로 했다  
근데 문제는... **뭘 어떻게 시작해야 할지 모르겠다는 것.**


그래서 Antigravity를 적극 사용해보기로 했다

---

## 💬 실제로 입력한 프롬프트들

### 1단계: 블로그 뼈대 만들기

```
"깃허브 블로그를 만들고 싶어. Jekyll 사용하고 싶어."
```

AI는 `_config.yml`, `index.html`, `_layouts/` 폴더 구조까지 알아서 만들어줬다.  
나는 그냥 **승인 버튼만 눌렀다.(딸깍)**

### 2단계: 디자인 요청

```
"디자인을 더 예쁘게 해줘. 깔끔하고 모던한 느낌으로."
"색상이 너무 밝아. (참고 이미지 url) 이런 느낌으로."
"이 부분은 Bold 처리해줘."
```

AI가 CSS를 작성하고, 카드형 레이아웃을 만들고, 호버 효과까지 넣어줬다.  
**나는 결과물을 보고 "좋아" 또는 "이건 바꿔줘"라고만 했다.(그저딸깍)**

### 3단계: 카테고리 구조 설계

```
"카테고리를 4개로 나누고 싶어. 
Bridge - 자기소개
Insight - 공부 기록  
Lab - 미니 프로젝트
Log - 개발 일지"
```

AI가 각 카테고리 페이지를 만들고, 네비게이션도 연결해줬다.

### 4단계: 첫 포스트 작성

```
"Lab 카테고리에 숫자 맞추기 게임 포스트를 작성해줘."
```

파이썬 복습을 할 겸, Claude에게 단계별 미니 프로젝트를 제시해달라고 했다.  
1단계인 숫자 맞추기 게임을 마침 완료해서, 업로드 해보기로 했다.   
들어갈 내용들과 내가 작성한 코드를 모두 넣었다.   
게임 코드와 설명까지 포함된 포스트가 뚝딱 완성됐다.

## 🎯 AI 활용 팁 (입문자 관점)

| 이렇게 말하면 | 결과 |
|-------------|------|
| "블로그 만들어줘" | 기본 구조 생성 |
| "더 예쁘게" | 디자인 개선 |
| "이건 마음에 안 들어" | 수정 제안 |
| "왜 안 돼?" | 원인 분석 + 해결 |

**핵심은 구체적으로 말할수록 좋다는 것!**

```
❌ "블로그 좀 고쳐줘"
✅ "홈페이지 카드 디자인에 그림자 효과 넣어줘"
```
